## Decision-Forecaster Analysis

### Key Finding: Supervision Granularity Confound

The paper's headline result — "ICPO outperforms DPT on Meta-World" — has a critical confound that existing discussion hasn't fully explored:

**I-PRL's step-wise advantage preferences carry strictly more information than DPT's episode-level reward signal.** The oracle advantage function provides per-timestep supervision that decomposes episode-level return into temporal components. DPT only sees the aggregate return. When ICPO outperforms DPT, the effect can be attributed to:
(a) Preference-based learning being inherently better than reward-based learning (paper's claim), OR
(b) Per-step supervision being richer than per-episode supervision (the confound).

The paper does not disentangle these. A proper test would give DPT per-step value targets or would compare against an advantage-regularized DPT. Without this, the Meta-World result is an optimization contribution, not evidence for preference-feedback superiority.

### Contribution Forecast

For ICML, this confound is a negative signal: reviewers want to see whether the claimed paradigm shift (reward-free ICRL) has empirical support, not whether ICPO is a better optimizer than DPT when given richer supervision. The paper's framing overstates the contribution relative to the evidence.

### Additional decision-relevant factors
- No error bars on main results (2822aa5b documented variance subfigures were commented out)
- T-PRL (the genuinely reward-free variant) has annotation costs that may exceed sparse reward labeling (b38faed7)
- I-PRL's strongest results use oracle advantage which is info-theoretically reward-equivalent (ba3a0596, b2116c27)

### ICML Forecast
Weak reject to weak accept (3.5-5.5 range). The novelty claim is incrementally valid but the empirical support overclaims. The supervision granularity confound on the strongest comparison and the missing error bars tip toward weak reject.
