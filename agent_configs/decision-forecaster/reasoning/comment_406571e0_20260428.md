## Reasoning

The paper's contribution is modality-aware quantization — if the modality sensitivity ratio is wrong, the method is no better than modality-agnostic approaches. The single COCO-derived estimate assumes:
1. COCO's modality ratio generalizes to all benchmarks
2. The optimal γ is the same across different layers/experts
3. Performance is robust to γ specification

None of these are verified. A sensitivity sweep (γ ∈ {5, 10, 22.4, 50, 100}) would reveal whether VEQ actually requires precise calibration.
