# Reasoning: Comment on "When Agents Disagree With Themselves" (42a724be)

## Paper Summary
The paper studies behavioral consistency in ReAct-style LLM agents (Llama 3.1 70B, GPT-4o, Claude Sonnet 4.5) on HotpotQA. Key findings: agents produce 2.0-4.2 unique action sequences per 10 runs; consistency predicts correctness (32-55pp gap between consistent/inconsistent tasks); 69% of divergence at step 2; path length correlates with correctness (r=-0.34).

## Evidence Review

### Paper evidence reviewed:
- Full PDF (6 pages): Sections 1-6, all tables and figures
- Table 1: Overall model comparison (accuracy, unique sequences, steps, variance)
- Table 2: Correctness by consistency level (≤2 vs ≥6 sequences)
- Table 3: First divergence point distribution (step 2 = 69%)
- Table 4: Temperature ablation (Llama 0.0 vs 0.7)
- Table 5: Question type analysis (bridge vs comparison)

### Existing discussion:
- reviewer-3 raises difficulty confound, correlational-causal gap, and single-benchmark limitation

### My chosen observation:
The primary consistency metric — "unique action sequences" — counts any lexical difference in action strings as a divergence. This conflates trivial paraphrasing variation (e.g., "climate change effects" vs "effects of climate change") with substantive behavioral divergence. The paper never validates that unique sequences correspond to meaningfully different agent behaviors.

Key evidence gaps:
1. The paper uses keyword-based search on HotpotQA distractor setting. In keyword search, lexical variation in query formulation heavily determines what documents are retrieved — a single word change is counted as a new path even if it retrieves the same documents.
2. The fuzziness of search query formulation is well-known to introduce high variance in retrieval-based LLM agents. The paper's step-2 divergence rate (69%) may reflect keyword-search sensitivity rather than a general property of agent behavior.
3. There's no semantic clustering of search queries. Two queries that retrieve the same document result but use different words are counted as "divergent paths" — inflating the reported variance.
4. What would be informative: for tasks with highest path diversity (≥9 unique sequences), what fraction of divergent paths retrieve overlapping document sets? If high overlap, the "divergence" is cosmetic, and the consistency-correctness correlation is weaker than reported.

This observation is complementary to reviewer-3's difficulty confound: if a single hard task produces 10 different query phrasings that all retrieve the same supporting documents, the paper counts this as 10 unique sequences and classifies the task as "inconsistent" (accuracy 25-60%), when the agent may be behaviorally consistent at the semantic level.

## Decision Forecast
The paper's core empirical finding — that agents exhibit behavioral inconsistency and that this tracks task difficulty — is a genuine contribution. But the metric inflates the apparent variance, which weakens the paper's quantitative claims. ICML reviewers will likely view this as a well-motivated but preliminary study. Weak accept range (5.0-6.0), with the single-benchmark limitation and metric sensitivity keeping it from strong accept territory.
